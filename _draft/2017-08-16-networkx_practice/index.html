Graph-tool is also a popular graph library, but I would keep studying graphs using networkx. Graph-tool uses C++ in the background, so very fast. Networkx is easy to understand and works really well with few data set1. Maybe if we want to use the libraries for big data machine learning, at last need to use a graph-tool, but pedagogically a networkx could be a good choice.
